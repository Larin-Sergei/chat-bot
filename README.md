# Chat Bot

Адаптивный чат-бот на HTML, CSS и JavaScript.

## Возможности

- Адаптивный интерфейс
- Сообщения снизу вверх
- Анимация набора сообщения
- Поддержка команд чат-бота
- Калькулятор математических операций
- Валидация ввода

## Команды

### Начало работы

/start

Бот начинает диалог с пользователем.

---

### Указать имя

/name: Вася

После ввода имени бот приветствует пользователя.

---

### Ввод чисел

/number: 7, 9

Бот сохраняет числа и предлагает выбрать операцию.

---

### Математические операции

+

-

*

/

После ввода операции бот выводит результат вычисления.

---

### Завершение работы

/stop

Бот завершает диалог.

---
